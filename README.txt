Automatic Court Order Generator v8 — Fixed Package

1. Extract the ZIP file completely.
2. Keep all five files in the same folder.
3. Open index.html in Google Chrome or Microsoft Edge.

Files:
- index.html — webpage
- styles.css — interface styling
- app.js — application logic
- order-varieties.js — embedded order templates
- jszip.min.js — local JSZip 3.10.1 library

The application remains client-side and retains the existing browser-storage key.
